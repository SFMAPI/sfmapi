# SDKs

Two officially-supported SDKs:

- **Python** — `pip install sfmapi-client`. See below.
- **TypeScript** — `npm install @sfmapi/client`. See
  [@sfmapi/client](https://www.npmjs.com/package/@sfmapi/client) and
  [the TS SDK README](https://github.com/sfmapi/sfmapi/tree/main/clients/typescript).

Both speak the same `/v1` REST API and share the same model shapes.
